Hotel Management

A project that analyzed and presented hotel data using MS Excel, SQL Server and PowerBI.

Project Description

The company runs two major hotel brands – City Hotel and Resort Hotel – in multiple locations across major cities around the world. The leadership wanted information that would help them answer the following questions:

1. Is our hotel revenue growing by year?

2. Should we increase our parking lot size?

3. What trends can we see in the data?

They wanted this information to be consolidated on a single dashboard for easy access and subsequent reference. The company provided hotel data for the different hotels from 2018 to 2020. When combined into a single database, the dataset had almost one million rows of information.

Method and Tools Used

The following steps and tools to deliver the solution:

1. I converted the datasets to comma separated value (.csv) files using MS Excel. This was done to enable me use the data in an SQL (Structured Query Language) environment.

2. I imported the data into SQL Server where I built a database and transformed the datasets to aid my analysis.

3. Still in SQL Server I performed the requisite analysis by using SQL to query the dataset and exporting the relevant results back to Excel.

4. I loaded the results into PowerBI and adjusted the data structure by creating new columns and formatting others to get my final dataset.

5. Still in the PowerBI environment, I created the dashboard as seen above and added important features such as slicers, dropdown menus, and trend lines to make the dashboard interactive.

6. I exported the final dashboard and shared with the principals of the hotel company.

Challenges

The primary challenge during this project was the issue of missing data points. This meant conferring with the company’s stakeholders in a series of back and forth to get clarifications and make necessary adjustments to the dataset.
